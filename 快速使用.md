# 快速使用指南

5 分钟上手跨境电商选品 AI Agent。

## 1. 安装 uv（一次性）

**Windows**（PowerShell）：
```powershell
powershell -c "irm https://astral.sh/uv/install.ps1 | iex"
```

**Mac**（终端）：
```bash
curl -LsSf https://astral.sh/uv/install.sh | sh
```

安装后**重启终端**，验证：`uv --version`

## 2. 安装项目依赖

```bash
cd aiagent
uv sync
```

**Windows 用户**：以上命令在 PowerShell 或 CMD 中均可运行。

## 3. 配置 AI（可选）

不配置也能用，系统自动生成本地模板报告。

```bash
cp .env.example .env
```

编辑 `.env`，填入 DeepSeek API Key：

```
DEEPSEEK_API_KEY=sk-你的key
DEEPSEEK_BASE_URL=https://api.deepseek.com
DEEPSEEK_MODEL=deepseek-v4-flash
```

## 4. 启动

### 一键启动（推荐）

| 系统 | 方式 |
|------|------|
| **Windows** | 双击 `run.bat` |
| **Mac** | 双击 `run.sh`（或终端 `./run.sh`） |

### 手动启动

```bash
uv run streamlit run app.py
```

浏览器打开 http://localhost:8501

## 5. 使用

### 方式一：示例数据（推荐首次体验）

1. 点左侧 **「使用示例数据」** 按钮
2. 市场选 `US`，类目填 `Pet Supplies > Dog Bowls`
3. 点 **「开始分析」**
4. 查看评分、报告，下载结果

### 方式二：上传自己的数据

1. 用卖家精灵导出 Amazon 类目 Top 100 表格（CSV 或 XLSX）
2. 在左侧上传文件
3. 选市场、填类目
4. 点 **「开始分析」**

## 6. 看什么

| 内容 | 说明 |
|------|------|
| 数据预览 | 前 10 行清洗后的数据 |
| 基础指标 | 价格、评分、评论统计 |
| 价格带分布 | 柱状图 |
| 品牌集中度 | Top 品牌占比 |
| 价格机会 | 主流价格带 + 建议切入区间 |
| 风险检测 | 匹配的风险关键词 |
| 综合评分 | 0-100 分 + 等级 |
| 产品机会 | 低评分高评论候选产品 |
| 选品报告 | 12 章节 Markdown |
| 下载 | .md 报告 + 清洗后 .csv |

## 7. 理解评分

```
>= 80 分 → 优先开发
65-79 分 → 进入二次验证
50-64 分 → 谨慎观察
< 50 分  → 不建议开发
```

## 8. 注意事项

- 分析结论为选品**辅助参考**，不能替代人工验证
- 不做自动爬虫，需手动从卖家精灵导出数据
- 不含 API Key 也能用基础功能
- Windows 首次启动如被防火墙拦截，点「允许访问」
- 端口 8501 被占用时，Streamlit 会自动换端口，看终端提示
